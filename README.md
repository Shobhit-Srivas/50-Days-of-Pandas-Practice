# 🐼 50 Days of Pandas Practice

A complete collection of **50 Pandas Interview and Practice Questions** designed for beginners and intermediate learners.

This repository helps you strengthen your Pandas skills through hands-on examples covering:

* 📊 DataFrame Operations
* 🧹 Data Cleaning
* 🔍 Data Filtering
* 📈 GroupBy & Aggregation
* 📋 Pivot Tables
* 🏷️ Data Transformation
* 📉 Missing Value Handling
* 🎯 Interview Preparation

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/50-Days-of-Pandas-Practice.git
```

Navigate to the project directory:

```bash
cd 50-Days-of-Pandas-Practice
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
